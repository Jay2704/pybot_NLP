**PyBot** is a full-stack Python programming Q&A assistant. It retrieves answers from **Stack Overflow–derived data** using **TF–IDF vectorization** and **cosine similarity**, exposed through a **FastAPI** backend and a **React** chat UI—deployed here as a single **Docker** service on port **7860**.
